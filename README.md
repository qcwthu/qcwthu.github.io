# Chengwei Qin's Personal Website

This repository builds the personal website at <https://qcwthu.github.io>.

## Main Content

- `_pages/about.md`: homepage
- `_pages/publications.md`: selected publications
- `_pages/students.md`: students page
- `_data/navigation.yml`: top navigation
- `_data/students.yml`: current and former student lists
- `_config.yml`: site-wide settings and author profile

## Run Locally

```bash
bundle install
bundle exec jekyll serve
```

The local site is served at <http://localhost:4000>.
